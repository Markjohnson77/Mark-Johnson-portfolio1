FULL CASE STUDY

CollegeGO – Smart Bus Pass & Movement Approval System

Overview

CollegeGO is a web-based platform that digitizes student movement requests and bus pass approvals through a structured multi-level workflow. It replaces manual paperwork with a secure, role-based approval system.

The Problem

Bus pass approvals in colleges are often handled manually, leading to delayed approvals, lack of transparency, and administrative overload. Students often face uncertainty regarding their request status while administrators manage physical forms.

The Solution

A digital approval workflow: Student → HOD → Warden. Providing real-time tracking, status-based dashboards, and paperless record management.

My Role

  • UI/UX Design
  • Frontend Development
  • Backend Integration
  • Database Design

Key Features

  • Role-based Access
  • Multi-stage Approvals
  • Real-time Tracking
  • Responsive UI

Tech Stack

AI-POWERED WORKFORCE PLATFORM

NOVA HR – Intelligent Workforce Management System

Overview

NOVA HR is an AI-powered workforce management platform that streamlines HR operations, employee management, and performance tracking in a single unified system.

The Problem

Traditional HR systems are fragmented, manual, and inefficient, leading to delayed workflows, lack of transparency, and poor employee experience with no real-time insights.

The Solution

A centralized, role-based HR platform that automates workflows, provides real-time dashboards, enables data-driven decisions, and improves employee self-service.

My Role

  • UI/UX Design (Figma)
  • Frontend Development
  • Backend Integration
  • Database Design

Key Features

  • Role-based Access (HR & Employee)
  • Operations Board (Task Tracking)
  • Performance Analytics (KPI)
  • Employee Management
  • Attendance & Leave System
  • Recruitment Module
  • Growth & Learning System

Design Approach

Dashboard-first UX for quick insights, featuring a clean, minimal UI with strong hierarchy and a color-coded system for clarity and reduced cognitive load.

Tech Stack

Frontend

  • HTML5 / CSS3
  • JavaScript
  • Figma

Backend

  • PHP
  • MongoDB

Impact & Learnings

Achieved faster HR workflows and improved transparency. Key learnings included the criticality of role-based UX and that data presentation matters more than quantity.
